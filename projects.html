zeri

This Discord bot programming using Node.js converts a date and time to the sexagenary cycle used for time recording in East Asia. The sexagenary cycle is used in names for historical events, such as the Imjin War and the Xinhai Revolution, and also has applications in astrology and fortune telling as well.


numc

A C program that speeds up matrix multiplication through the use of parallelism and SIMDs.
